RedisLongStore

class RedisLongStore extends ConvertedStore[ChannelBuffer, ChannelBuffer, ChannelBuffer, Long] with MergeableStore[ChannelBuffer, Long]

A MergableStore backed by redis which stores Long values. Values are merged with an incrBy operation.
